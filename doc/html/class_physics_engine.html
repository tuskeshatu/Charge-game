Detailed Description

Represents a physics engine for simulating physical forces and collisions.

Represents a physics engine for simulating physical interactions.

The PhysicsEngine class provides functionality for calculating electric forces, friction forces, and checking collisions. It also allows updating the player's physics state.

The PhysicsEngine provides interface for simulating forces.
