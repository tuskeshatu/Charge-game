Detailed Description

Represents a player in the game.

The Player class inherits from the Charge class and represents a player in the game. It contains information about the player's speed, mass, and body shape.

The Player class encapsulates the properties and behaviors of a player in the game. It provides methods to interact with the player, such as setting the player's name, getting the player's score, and updating the player's position.
